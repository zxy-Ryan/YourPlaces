# YourPlaces
This is a MERN stack demo web application.  
Users can share places (with image and location) with other users.  

**Functions**:  
* Allow users to see shared places of all users, no matter if users are logged in. 
* Control which functions are available to users by authentication and authorization.
* Allow users to do CRUD (Create, read, update and delete) operations on their own places.  

URL: https://your-places-sharing.web.app/
